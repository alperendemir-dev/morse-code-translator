# Morse Code Translator (C#)

This is a console-based application for translating text into Morse code and Morse code back into text.

## Features
- Translate text to Morse code
- Translate Morse code to text
- User input handling
- Console-based interaction

## Technologies
- C#
- .NET Console Application

## Purpose
This project was created to practice string handling, conditional logic, and console-based programming.
